# 📈 Stock Market Analysis Dashboard | Power BI

An interactive Stock Market Analysis Dashboard built using **Power BI** to analyze the latest 100 days of stock performance for **Apple Inc.** and **IBM Inc.** The dashboard uses API-based data, Power Query for data transformation, and DAX for KPI calculations to provide meaningful business insights.

---

## 📌 Project Overview

This project provides a comprehensive analysis of Apple and IBM stock market performance. It enables users to monitor stock prices, trading volume, and market trends through interactive visualizations and dynamic filtering.

---

## 🚀 Features

- Live Stock Data using API Integration
- Data Cleaning & Transformation using Power Query
- Dynamic KPI Cards
- Interactive Slicers (Date & Month)
- Monthly Stock Price Trend Analysis
- Trading Volume Analysis
- High vs Low Price Comparison
- Apple & IBM Dashboard Navigation
